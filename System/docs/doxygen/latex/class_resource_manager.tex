\subsubsection*{template$<$typename Resource$>$ class ResourceManager$<$ Resource $>$}

Provides the base functionality for Resource caching and access. 

The \doxyref{ResourceManager}{p.}{class_resource_manager} gives some basic functionality to aid in resource management. It gives the user a basic method of caching, accessing, and clearing resources from the system. Loading a resource and registering it with the manager is left to the subclasses. \begin{Desc}

Register a new resource with the system. A name is not necessarily needed, but if none is given then there will be no way to again retrieve the Resource from here. \begin{description}
